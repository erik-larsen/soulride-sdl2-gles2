Drop music files here (.ogg, .mp3, .wav, or .flac) and Soul Ride will
play them as the soundtrack, shuffled, like the original CD audio.

Optional: a cdaindex.txt in this directory provides track captions,
same format as the original CD index (song / artist / two comment
lines per track, blank line between tracks).

Set MusicPath in config.txt to use a different directory.
